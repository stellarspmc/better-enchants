## Mod Support

### Performance Mods
- [x] Sodium (v0.8.12 / v0.6.13)
- [x] ImmediateFast
- [x] ModernFix
- [x] Lithium (Shouldn't have impact, it optimizes server stack)
- [x] FerriteCore
- [x] Model Gap Fix
- [x] Particle Core 

> Safe to assume that most performance mods are supported.
